**Snake Game**  
This is a multiplayer snake game where players compete against each other to earn the highest score   
all while avoiding walls, other snakes, and even their own tail. This project serves as a fun way 
to interact and compete with friends. It also features a web server that contains and displays information about 
your game, such as players, high scores, and enter and leave times.  
  
**Features**  
- Classic snake gameplay
- Snakes grow upon obtaining a power-ups, which spawn randomly
- Snakes die upon collisions with walls, other snakes, or their own tail
- Live scoreboard
- Smooth controls
- Respawn after death
  
**Installation**  
(_Note: Git and Net.9 required_)  
  
To install this project, simply clone the repository:  
  - git clone https://github.com/crsheehan/SnakeGame.git
  - cd SnakeGame\SnakeLauncher
  - dotnet run  
  
_Important Note: Snake Server and Connection to database require the local machine to be on the University of Utah wifi to run.  _
 

**Known Bugs**  
- Sometimes when building a project first time, it fails. Try running it again  
- I have had an issue with  the snakes being invisible
- Sometimes, when joining the server, it crashes. Try refreshing the page and try again
